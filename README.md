## 📌 Notable Work

* **[Multi-Agentic RAG Interview System](https://github.com/pranava-kp/Final-Year)** – Orchestrated specialized LLM agents using LangChain and LangGraph for adaptive, resume-tailored technical interviews. Integrated a closed-loop evaluation cycle with Pinecone vector retrieval to deliver real-time feedback.

Implemented dynamic staff filtering via APIs before successfully handing the project over to a new maintainer.
